## Barcode scanner compatibility

Tested with Symbol LS2208 (USB) and Socket Mobile S700 (Bluetooth).
Both work in keyboard wedge mode. Code 128 barcode format recommended
for alphanumeric SKUs. QR codes as fallback for mobile phone scanning
using device camera.
