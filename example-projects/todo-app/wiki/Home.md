## Project Overview

TaskFlow is a personal task management web app built for Acme Consulting. The system helps consultants manage their workload while giving managers visibility into team capacity.

## Key Features

- Personal task lists for each consultant
- Quick task creation (under 30 seconds)
- Team workload view for managers
- Mobile-friendly for on-site work

## Task States

Tasks flow through these states:
1. **Todo** - Not yet started
2. **In Progress** - Currently being worked on
3. **Done** - Completed

## User Roles

- **Consultant** - Manages their own tasks
- **Team Lead** - Can view all team members' tasks
